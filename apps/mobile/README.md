# CodeQuest Mobile

Application mobile React Native / Expo.

## Stack Prevue

- React Native
- Expo
- TypeScript
- Zustand
- TanStack Query

## Objectif Initial

Construire le MVP mobile avec des donnees locales:

- tableau de bord;
- mission du jour;
- entrainement;
- resultats;
- erreurs a revoir;
- profil conducteur.
