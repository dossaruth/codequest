# Roadmap CodeQuest

## Phase 0 - Fondation

- Choisir le nom du projet
- Definir la vision produit
- Definir l'architecture cible
- Creer la structure du repository
- Rediger le README initial

## Phase 1 - MVP Mobile Local

Objectif: creer une premiere application utilisable sans backend.

- Ecran d'accueil avec progression fictive
- Profil conducteur local
- XP et niveau
- Mission quotidienne statique
- Liste de themes
- Serie de questions locale
- Correction simple
- Historique d'erreurs local

## Phase 2 - Backend Principal

Objectif: connecter l'application a une API Spring Boot.

- Projet Spring Boot Java
- Authentification JWT
- Gestion utilisateur
- Profil conducteur
- Themes
- Questions
- Sessions d'entrainement
- Resultats
- Migrations Flyway
- PostgreSQL via Docker

## Phase 3 - Progression Et Gamification

- Calcul XP
- Niveaux
- Rangs conducteur
- Badges
- Succes
- Serie de connexion
- Challenges
- Tableau de bord personnalise

## Phase 4 - Revision Intelligente

- Statuts d'erreur: non comprise, a revoir, maitrisee
- Repetition espacee
- Mode faiblesse
- Missions quotidiennes dynamiques
- CodeQuest Score initial

## Phase 5 - Coach IA

- Microservice FastAPI
- Analyse des erreurs

- Explications pedagogiques
